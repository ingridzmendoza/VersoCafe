# Verso Café

Verso Café es una página web que simula el sitio de compra y venta de un café que también funciona como espacio de lectura y librería, ya que ofrece libros a la venta.
El proyecto está enfocado en la experiencia del usuario y en la simulación de un flujo completo de navegación y compra.

Este sitio no procesa pagos reales; su objetivo es simular el funcionamiento de una plataforma de comercio electrónico.

Está hosteada en GitHub Pages: [Verso Cafe](https://ingridzmendoza.github.io/VersoCafe/)

## Descripción del proyecto

Verso Café representa un espacio digital donde los usuarios pueden:
- Comprar café y libros
- Guardar productos en favoritos
- Simular una compra mediante un carrito
- Cambiar entre modo claro y modo oscuro
- Cambiar el idioma de la interfaz (español / inglés)
 
 
La interfaz inspirada en un café literario.


## Funcionalidades principales

- **Listado de productos**
  - Cafés y libros
  - Visualización de productos más populares
  - Ordenamiento por popularidad y precio (ascendente y descendente)

- **Tabs de navegación**
  - Visualización de todos los productos
  - Visualización exclusiva de cafés
  - Visualización exclusiva de libros

- **Favoritos**
  - Guardado de productos como favoritos
  - Persistencia por usuario utilizando `localStorage`

- **Carrito de compras**
  - Agregar productos al carrito
  - Simulación de compra
  - Persistencia del carrito en `localStorage`

- **Autenticación simulada**
  - Inicio de sesión mediante correo electrónico
  - Carga de favoritos y carrito del usuario autenticado

- **Modo claro / oscuro**
  - Cambio dinámico de tema
  - Preferencia guardada en `localStorage`

- **Cambio de idioma**
  - Español / Inglés
  - Textos dinámicos según el idioma seleccionado


## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES Modules)
- Bootstrap
- LocalStorage
